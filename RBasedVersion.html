Running the course using RStudio
(and GitHub)

The course is best run using RStudio and GitHub (so that all course material can also be shared and reused).

We will be using RStudio for data analytics and github to share and publish course material and class projects. All participants are required to setup a github account and provide their username before the first session.



Using RStudio

Download and install RStudio. You can also browse through some of the tutorials available in that site.

Browse through the course technical resources as starting points. Moreover, these are some more instructions you will find useful.



Using GitHub

Create a github account before session 1 and share your username with us.

Browse through the course technical resources related to GitHub.
